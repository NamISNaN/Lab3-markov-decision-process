马尔科夫决策过程

马尔科夫随机过程（MDP, Markov Decision Process）是一种离散时间随机控制过程。

在每一步中，随机过程处于状态 sss，决策者此时采取一个可行的决策 aaa。在下一步随机过程将转移到新的状态 s′s's′，并给决策者相应的回报 Ra(s,s′)R_a(s, s')Ra​(s,s′)。

随机过程进入新状态 s′s's′ 的概率取决于所采取的决策。具体来说，其概率大小有转移矩阵中的 Pa(s,s′)P_a(s, s')Pa​(s,s′) 定义。因此，下一个状态 s′s's′ 取决于当前状态 sss 和决策者采取的决策 aaa。在已知 sss 和 aaa 的情况之下，状态 s′s's′ 与所有之前的状态都是（条件）独立的。也就是说 MDP 具有马尔科夫性质，即随机过程后续的状态的条件概率分布只与当前状态有关，与之前的状态无关。

定义

马尔科夫过程是一个 5 元组 (S,A,Pa,Ra;γ)(S, A, P_a, R_a; \gamma)(S,A,Pa​,Ra​;γ)，其中：

  • SSS 是有穷状态空间集合
  • AAA 是有穷决策集合，AsA_sAs​ 表示状态 sss 下可以采取的决策集合
  • Pa(s,s′)=Pr(st+1=s′∣st=s,at=a)P_a(s, s') = Pr(s_{t+1} = s' | s_t = s, a_t = a)Pa​(s,s′)=Pr(st+1​=s′∣st​=s,at​=a) 表示在 ttt 时刻处于状态 sss 并采取决策 aaa 将在 t+1t + 1t+1进入状态 s′s's′ 的概率，此概率与时间 ttt 无关
  • Ra(s,s′)R_a(s, s')Ra​(s,s′) 是从状态 sss 经过决策 aaa 到达状态 s′s's′ 时的期望瞬时收益
  • γ∈[0,1)\gamma \in [0, 1)γ∈[0,1) 是决策因子，表示将来状态与当前状态的重要度差别

问题

假定决策的方案：一个函数 π\piπ，其中 π(s)\pi(s)π(s) 表示在状态 sss 下采取的决策。一旦有了决策方案 π\piπ，再联合在给定方案下到达下一状态的概率 Pr(st+1=s′∣st=s,at=a)Pr(s_{t+1} = s'|s_t = s, a_t = a)Pr(st+1​=s′∣st​=s,at​=a)，我们可以得到到达下一状态的概率 Pr(st+1=s′∣st=s)Pr(s_{t+1} = s'|s_t = s)Pr(st+1​=s′∣st​=s)，这就是一个马尔科夫转移矩阵。

MDP 的核心问题在于找到最优的决策方案。我们的目标是选择最优的 π\piπ，使得期望收益最大化，通常我们会将将来的收益打一个折扣，因为无穷远的时间收益即使很大也没有意义：

(1)W=∑t=0∞γtRat(st,st+1)这里 at=π(st)，也就是我们采取的决策方案W = \sum_{t=0}^\infty \gamma^tR_{a_t}(s_t, s_t+1) \quad \text{这里 $a_t = \pi(s_t)$，也就是我们采取的决策方案} \tag{1} W=t=0∑∞​γtRat​​(st​,st​+1)这里 at​=π(st​)，也就是我们采取的决策方案(1)

其中 γ\gammaγ 用来表示折扣因子并且满足 γ∈[0,1)\gamma \in [0,1)γ∈[0,1)。例如，当折扣率为 rrr 时，即后一时刻 1+r1+r1+r 块钱的收益只相当于当前时刻 111 块钱的收益，那么就有 γ=11+r\gamma = \frac{1}{1+r}γ=1+r1​。

由于 MDP 具有马尔科夫性质，WWW 的值与时间是无关的，因此其只是 sss 的函数。

随机需求的单商品存贮决策

问题描述

每个月，仓库经理都会清点某种商品的当前库存量，从而决定是否要从供应商那里进货，进货的话要进多少。在此过程中，他需要权衡该商品库存带来的成本，和不能满足消费者对该商品的需求所带来的损失。他的目标就是最大化各月所得收益和期望值。我们设商品的需求量是一个已知概率分布的随机变量，且积压订单是不允许的，故库存量不会为负数。

  • sts_tst​ 是第 ttt 个月的初库存量，它是状态变量
  • ata_tat​ 是第 ttt 个月的订货量，它是决策变量
  • DtD_tDt​ 是第 ttt 个月的随机需求量，假定该需求满足一个时间齐次的分布 pj=p(Dt=j),j=0,1,2...p_j = p(D_t = j), j = 0, 1, 2...pj​=p(Dt​=j),j=0,1,2...，也就是说需求量的分布与时间 ttt 无关

由于库存量非负，得到状态转移方程：

(4)st+1=max{st+at−Dt,0}≡[st+at−Dt]+s_{t+1} = max\{s_t + a_t - D_t, 0\} \equiv [s_t + a_t - D_t]^+ \tag{4} st+1​=max{st​+at​−Dt​,0}≡[st​+at​−Dt​]+(4)

假设：

  • 每个月月初做出是否订货和订货数量的决策，并假定订货可以及时送到
  • 对商品的需求贯穿整个月，但是在该月的最后一天所有订单必须得到满足
  • 如果顾客对某商品的需求超过该商品的库存量，即顾客的需求得不到满足，顾客可以到别处去购买他所需的商品。因此不会有因供货不足而造成订单积压的问题
  • 收益、成本和需求分布不会按月改变
  • 产品售出量都是整数
  • 仓库容量为 MMM 个单位
